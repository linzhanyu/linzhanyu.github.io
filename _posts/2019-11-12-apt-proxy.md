随着神经网络技术应用到伟大的防火墙上以后，墙越来越聪明。终于 apt 源也被盯上了。

现以 Ubuntu 系统为例：

---


#### 1. 仅命令行一次性使用代理：

{% highlight shell %}
sudo apt -o Acquire::socks::proxy="socks5://127.0.0.1:1080/" update
{% endhighlight %}

---

#### 2. 一次配置永远享受

将下面一行内容

{% highlight shell %}
Acquire::socks::proxy "socks5://127.0.0.1:1080/";
{% endhighlight %}

写入文件

{% highlight shell %}
/etc/apt/apt.conf.d/12proxy
{% endhighlight %}

# 当GRUB遇到高分辨率显示屏

- 出现的问题 : 文字小，并且反应迟缓

解决方法：让GRUB工作在一个合适的分辨率下

编辑文件
``` bash
vim /etc/default/grub 
```

加入以下内容

``` conf
GRUB_GFXMODE=1280x1024
```

然后更新 GRUB

``` bash
sudo update-grub
sudo reboot
```

## 一行命令在Python3中装好Phoenix

{% highlight shell %}
> sudo python3.4 -m pip install --upgrade --pre -f https://wxpython.org/Phoenix/snapshot-builds/ wxPython -b /mnt/disk/pip_build
{% endhighlight %}
